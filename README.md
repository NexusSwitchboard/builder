# Nexus Builder

Provides some convenience commands for operating on multiple Nexus projects at once.

## Requirements

* Python 3.7+
* pipenv

## Install

`pipenv install`

## Usage

### List
List all the Nexus projects in the directory above this one.

### Commit 
Commit all changes in the Nexus projects in the directory above this one

### Push 
Push all changes in the Nexus projects in the directory above this one


## Important

This is purely for convenience of developing locally on my specific machine.  Over time, if it proves 
useful, I will improve to be more configurable.
